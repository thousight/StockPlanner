## Verification Criteria
- [ ] `SupervisorAgent` correctly routes math-heavy queries to `code_generator`.
- [ ] `CodeGenerator` runs in parallel with other researchers (if fanned out).
- [ ] `AnalystAgent` successfully identifies and uses results wrapped in `<calculation_result>`.
- [ ] The graph correctly loops back from Analyst to Code Generator if a correction is requested.
- [ ] The loop terminates after 2 Analyst-triggered retries.

## must_haves
- All routing to `code_generator` must be logged in `agent_interactions`.
- The `code_revision_count` must be correctly incremented during loops.
- Feedback from the Analyst to the Generator must be specific and helpful.
